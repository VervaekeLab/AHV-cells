This is the main program and the modules that were used for controlling the setup during experiments.

The VIs were created and run in LabVIEW 2013.
Some VIs also depend on the Vision Acquisition 2015 toolbox

The TML_LIB_LabVIEW is needed to run the SteinMeyer DT240 rotary stage. An earlier version used a rotary stage from PI (Physik Instrument) and some code dependencies might still exists.
